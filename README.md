# 🌐 GlobalPulse: Live Global News Aggregator

GlobalPulse is a modern, real-time news aggregator designed to bring you the latest headlines from across the globe. Built as a feature-rich Progressive Web App (PWA), it provides a seamless and clutter-free reading experience across all devices.

## 🌟 Key Features

- **Global & Regional Coverage:** Switch between global headlines or specific news from Bangladesh, USA, UK, India, Canada, and Australia.
- **Real-Time Updates:** Fetches the latest stories directly from authoritative RSS feeds using the RSS2JSON API.
- **Immersive Reader Mode:** Enjoy a distraction-free environment for deep reading.
- **Instant Summaries:** Get the core facts quickly with an integrated summarization tool.
- **Audio Support:** Listen to articles on the go with built-in text-to-speech.
- **Personalized Bookmarks:** Save your favorite articles to a local library for later viewing.
- **Adaptive Dark Mode:** Sleek dark theme toggle for comfortable night-time reading.
- **PWA Powered:** Installable on mobile and desktop with offline support via Service Workers.
- **Reading Progress Bar:** Visual indicator to track your reading depth in every article.

## 🚀 Tech Stack

- **Frontend:** HTML5, Vanilla CSS3 (Custom Properties), JavaScript (ES6+)
- **Data Source:** RSS2JSON API (aggregating multiple global RSS feeds)
- **PWA:** Manifest.json & Service Workers for offline reliability and installation.

## 📸 Project Preview

The application features a responsive grid layout, interactive sidebars for regional navigation, and a sophisticated reader view with mini-sidebars for "Up Next" content.

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/GlobalPulse.git
   ```
2. **Launch the App:**
   Simply open `index.html` in any modern web browser.
3. **PWA Installation:**
   To enable PWA features (installing as an app), serve the project using a local server (like VS Code's *Live Server* or `npx serve`).
